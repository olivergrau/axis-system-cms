# AXIS Research Layer

This directory contains the notebook-oriented research layer built on top of
the core AXIS CMS framework.

Recommended structure:

- `context.md`
  research motivation, current state, and methodological framing
- `jupyter-structure.md`
  notebook roadmap and build order
- `lib/`
  shared helpers for loading artifacts, plotting, and notebook setup
- `notebooks/`
  the actual research notebooks
- `data/`
  optional copied or curated analysis data
- `figures/`
  exported notebook figures

The intent is to keep notebooks focused on explanation and interpretation
rather than repeated artifact-loading boilerplate.
