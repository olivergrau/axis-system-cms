# Neural Submodules in AXIS

This initiative explores how neural networks could be integrated into AXIS
systems as bounded submodules rather than as monolithic end-to-end agents.

The goal is not immediate usefulness or performance. The goal is to test:

- where neural components fit naturally into the existing mechanistic systems
- how training and inference should be separated or combined
- which integrations remain compatible with AXIS design assumptions
- which experimental directions are worth prototyping first
