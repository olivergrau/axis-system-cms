# Specifications

Implementation briefs (work packages) for every component of the AXIS framework.
Each brief specifies inputs, outputs, acceptance criteria, and test requirements.

## Core Framework

| Group | Work Packages | Scope |
|---|---|---|
| **WP-0.x** | Scaffold & Infrastructure | Package layout, test setup |
| **WP-1.x** | SDK | Protocols, world contracts, config types |
| **WP-2.x** | System A & World | World extraction, action engine, System A conformance |
| **WP-3.x** | Framework | Registry, runner, executors, persistence, CLI |

## Visualization Layer

| Group | Work Packages | Scope |
|---|---|---|
| **WP-V.0.x** | SDK Extensions | Trace types, world metadata |
| **WP-V.1.x** | Core | Supporting types, adapter protocols, registry |
| **WP-V.2.x** | Adapters | Grid2D, toroidal, signal landscape, System A, System B |
| **WP-V.3.x** | Replay | Access, snapshot resolver, playback, ViewModel |
| **WP-V.4.x** | UI | Canvas, overlays, panels, main window |
| **WP-V.5.x** | Tests | Adapter, replay, and end-to-end test suites |
