# Workspace Examples

This bootstrap note no longer points to a single "current" workspace.

To see real workspace examples, browse the repository's `workspaces/`
directory.

That folder contains example and active workspace layouts, including:

- single-system investigation workspaces
- system-comparison workspaces
- their manifests, configs, results, comparisons, and notes

Recommended approach:

1. Start with the workspace folder that matches your task.
2. Read its `workspace.yaml` manifest first.
3. Then inspect its `configs/`, `README.md`, and `comparisons/` as
   needed.

Primary location:

- `workspaces/`
