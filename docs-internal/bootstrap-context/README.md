# AXIS Context Pack

This directory contains short internal context files meant to help a new
agentic session establish working context quickly without re-analyzing the
entire repository.

These files are not intended to replace the main docs in `README.md`,
`docs/`, or `docs-internal/`. They are compressed orientation notes derived
from the current codebase and documentation.

Recommended reading order for a fresh session:

1. `01-solution-overview.md`
2. `02-codebase-map.md`
3. `03-execution-and-artifacts.md`
4. `04-current-workspace-system-aw.md` if the session is related to the
   current `system_aw` investigation workspace

Authoritative references still live in:

- `README.md`
- `docs/manuals/axis-overview.md`
- `docs/manuals/workspace-manual.md`
- `docs/manuals/comparison-manual.md`
- `docs/manuals/system-aw-manual.md`
- `docs-internal/architecture/`

These context notes reflect the repository state as observed on 2026-04-22.
