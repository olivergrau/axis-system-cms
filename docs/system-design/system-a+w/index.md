# System A+W

System A+W extends the baseline System A with a curiosity drive, a minimal spatial world model, and dynamic drive arbitration.

## Scope

System A+W introduces:

- dual-drive interaction,
- spatial novelty,
- sensory novelty,
- and a minimal visit-count world model.
