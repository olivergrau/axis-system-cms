# System B

System B is a scout agent operating on a dynamic signal landscape with a scan action and a monotonic energy budget.

## Documents

| Document | Content |
|---|---|
| [Formal Model](01_System B Model.md) | Full model: scan action, signal landscape, and weight computation |

## Scope

System B demonstrates:

- a non-drive-based agent architecture,
- signal-guided movement,
- read-only scanning,
- and behavior without resource extraction.
