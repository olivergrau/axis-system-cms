# System A

System A is the AXIS baseline system: a minimal mechanistic forager with a hunger drive, local perception, and no world model.

| Document | Content |

## Scope

System A defines the baseline architecture for:

- energy-driven behavior,
- local observation,
- single-drive action modulation,
- and softmax-based action selection.
