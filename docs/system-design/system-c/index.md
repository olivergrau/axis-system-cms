# System C

System C extends the AXIS baseline with local predictive memory, signed prediction error, and bounded action-level predictive modulation.

## Documents

| Document | Content |
|---|---|
| [Formal Model](01_System C Model.md) | Full model: predictive memory, signed prediction error, and bounded action-level modulation |

## Scope

System C introduces:

- a shared predictive feature space,
- context-conditioned expectation memory,
- positive and negative predictive traces,
- and predictive reshaping of action preferences.
