# Decisions and Postponements

## Neural Submodules in AXIS

Status: `postponed`

Reason:

The current explicit prediction subsystem has not demonstrated robust value in
the existing built-in world family. Neuralization is therefore postponed until
world/system alignment is clarified and a useful explicit baseline is available.

Supporting logbook entry:

- [Prediction-A and the Built-In Worlds: A Mismatch, Not a Failure](../logbook/2026/2026-06-28-prediction-world-mismatch.md)

Operational consequence:

- no neuralization of prediction before explicit usefulness is demonstrated
- maintain the neural initiative as a conceptual research track, but not as the active implementation focus

## Current Main Initiative

Status: `active`

The current main initiative is [World-System Alignment](../initiatives/world-system-alignment/index.md), which now carries the prediction-versus-world analysis work.

Operational consequence:

- current work should focus on refinement options, fair test worlds, and possible non-neural predictor redesigns

## Prediction-A Retention Decision

Status: `kept as reference subsystem`

Reason:

Prediction-A should not be discarded just because it did not show benefit in the
current built-in worlds.

It remains valuable as:

- a worked explicit example of local predictive trust learning
- a comparison point for alternative prediction architectures
- a baseline for future, better-matched world classes

Operational consequence:

- treat Prediction-A as a reference design
- do not treat it as generally validated under the current world family
