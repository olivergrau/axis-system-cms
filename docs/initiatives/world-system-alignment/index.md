# World-System Alignment

Status: `active`
Phase: `analysis`
Objective: determine which world classes and system structures are well-matched,
and where the current explicit prediction system fails because the worlds do not
provide the right exploitable structure.
Current assessment: the existing explicit prediction system is valuable as a
reference design, but it appears poorly matched to the current deterministic
resource-world family.
Next step: derive world and system refinement options from the recorded prediction/world mismatch.
Tags: `prediction`, `worlds`, `system-design`, `alignment`, `analysis`
Last updated: `2026-06-28`

This initiative examines the fit between AXIS systems and AXIS worlds.

Its immediate focus is the current explicit prediction system and the question
of whether the existing resource worlds can meaningfully support it.
