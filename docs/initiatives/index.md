# Initiatives

Initiatives collect forward-looking design and research directions that extend
AXIS beyond the currently implemented systems. They are more exploratory than
the core manuals and tutorials, but they remain technically concrete and tied
to follow-up engineering work.

## Available Initiatives

| Initiative | Description |
|---|---|
| [World-System Alignment](world-system-alignment/index.md) | Current main initiative. Examines which world classes and system structures actually fit each other, starting with the mismatch between the current explicit prediction system and the built-in resource worlds. |
| [Neural Submodules in AXIS](neural-submodules/index.md) | Longer-horizon initiative on integrating neural submodules into AXIS systems. Currently postponed until explicit subsystem usefulness and world alignment are better understood. |
